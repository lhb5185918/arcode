# 90后童年互动房间游戏

这是一个使用Python和Arcade库开发的2D互动游戏，模拟90后童年回忆中的房间场景，包含丰富的交互元素。

## 功能特点

- 逼真的90年代房间场景
- 可交互的电视（开/关机，切换频道）
- 可操作的遥控器
- 更多交互对象正在开发中...

## 安装说明

1. 确保已安装Python 3.7+
2. 安装依赖：
   ```
   pip install arcade
   ```

## 运行游戏

```
python interactive_room_game.py
```

## 游戏操作

- 点击电视右下角的红色按钮可以开关电视
- 点击遥控器可以切换电视频道（电视开启状态下）
- 更多操作将在后续版本中添加
